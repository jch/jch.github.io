Dark mode and RSS

Bit of clean up for the blog. I've cleaned up the CSS to use browser defaults for the most part. Instead of using hex colors, I used named colors and chose colors that work well on a light and dark background. For dark mode, I used black for OLED screens instead of a dark gray.
